Curate YouTube content to my Notion recommendation database.

## Inputs

- YouTube channels in playground/youtube-notion/subscription.toml
- Notion Database ID and Auth key stored in .env.playground

## Your Task
Download the latest episode from each channels in the subscription list and put into the give Notion database. For each entry, summarize the video in the content body.

## Limitation
You are only allowed to use native bash commands plus the native tools (google search and url context) to solve this task. DO NOT attempt to write and execute Python code.
